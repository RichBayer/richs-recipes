# Rich's Recipes

Personal recipe collection for family favorites, smoking notes, sauces, rubs, holiday menus, and cook logs.

This repo is intentionally simple. Categories can grow naturally over time as recipes get added.

## Starting categories

- `bbq/` - smoked meats, grilling, barbecue recipes
- `sauces/` - vinegar sauces, BBQ sauces, marinades, dressings
- `rubs/` - dry rubs and seasoning blends
- `menus/` - holiday menus and event plans
- `cook-logs/` - notes from real cooks, timing, smoker behavior, lessons learned
- `main-dishes/` - everyday meals, family dinners, and scaled-down institutional recipes

## First cook

The first recipe set is for July 4, 2026: two smoked pork butts on the Pit Boss Brunswick vertical smoker, with a sweet Carolina-style vinegar sauce.

## Notes style

Each recipe should capture:

- What was cooked
- Equipment used
- Ingredients
- Process
- Timing
- What worked
- What to change next time
